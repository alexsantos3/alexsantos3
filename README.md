## Sobre mim

Sou um **Desenvolvedor Web em formação**, com foco na criação de **interfaces modernas, páginas responsivas, sistemas web e experiências digitais bem estruturadas**. Tenho experiência prática no desenvolvimento de **landing pages, interfaces, integrações e aplicações web**, sempre com atenção à **qualidade visual, performance, usabilidade e organização do código**.

Meu foco principal está em **React, Angular, Bootstrap, TypeScript, JavaScript e Node.js**, construindo soluções voltadas ao ecossistema web moderno.
Além de possuir **noções de PHP, Laravel, C# e ASP.NET**, o que fortalece minha visão sobre diferentes tecnologias, estruturas de projeto e arquiteturas de desenvolvimento.

Busco evoluir continuamente como desenvolvedor, transformando ideias em produtos digitais funcionais, bem construídos e com valor real para o usuário e para o negócio.
